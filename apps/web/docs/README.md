# Oxford 3000 Vocabulary Test - Web App

Веб-приложение для тестирования знания английских слов из списка Oxford 3000.

## Описание

Интерактивное приложение в стиле Tinder для проверки словарного запаса. Пользователь свайпает карточки со словами:
- **Вправо** - знаю слово
- **Влево** - не знаю слово
- **Вверх** - пропустить

## Технологии

- React 18 + TypeScript
- Vite (сборка)
- Ant Design (UI компоненты)
- jsPDF (экспорт в PDF)
- LocalStorage (сохранение прогресса)

## Установка и запуск

```bash

cd apps/web

npm install

npm run build

```

## Функциональность

### Выбор уровня
При старте пользователь выбирает уровень CEFR:
- **A1** - Beginner (базовая лексика)
- **A2** - Elementary (простые разговоры)
- **B1** - Intermediate (работа, путешествия)
- **B2** - Upper-Intermediate (сложные темы)
- **ALL** - Все уровни (полный список Oxford 3000)

### Управление

| Действие | Мышь | Клавиатура | Touch |
|----------|------|------------|-------|
| Знаю | Свайп вправо | → | Свайп вправо |
| Не знаю | Свайп влево | ← | Свайп влево |
| Пропустить | Свайп вверх | ↑ или Space | Свайп вверх |

### Прогресс
- Прогресс-бар показывает процент пройденных слов
- Статистика: знаю / не знаю / пропущено
- Автосохранение после каждого действия

### Экспорт PDF
Кнопка "Export PDF" создает документ со списком неизвестных слов для изучения.

### Сброс прогресса
Кнопка сброса с модальным окном подтверждения.

## Типы данных

```typescript
interface Word {
  id: number;
  word: string;
  pos: string;           // "n.", "v.", "adj."
  level: 'A1' | 'A2' | 'B1' | 'B2';

type LevelFilter = 'A1' | 'A2' | 'B1' | 'B2' | 'ALL';
```

## Алгоритмы

### Рандомизация
Используется алгоритм Fisher-Yates для перемешивания слов при старте теста.

### Пропущенные слова
Пропущенные слова не повторяются автоматически. Они сохраняются в отдельной категории для возможного повторного прохождения.
